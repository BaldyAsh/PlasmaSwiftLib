  • preparePlasmaContractWriteTx(method:value:parameters:extraData:)

    Prepares transaction that writes to some method of Plasma Contract in Ethereum Blockchain. It is broadcasted to the network, processed by miners, and if valid, is published on the blockchain, consuming Ether.

    Throws

    Web3Error.transactionSerializationError if there is some errors in serialization

    Declaration

    Swift

    public func preparePlasmaContractWriteTx(method: PlasmaContractMethod = .deposit,
                                         value: BigUInt = 0,
                                         parameters: [AnyObject] = [AnyObject](),
                                         extraData: Data = Data()) throws -> WriteTransaction

    Parameters

    method

    method of Plasma Contract
    value

    Wei amount that is sending to Plasma Contract
    parameters

    an array corresponding to the list of parameters of the Plasma Contract method. Can be other arrays or instances of String, Data, BigInt, BigUInt, Int or EthereumAddress
    extraData

    additional data specific for used Plasma Contract method - it is appended to encoded parameters

    Return Value

    Ethereum transaction object

    write type transaction
    options

    additional options for sending that transaction: to, from adresses, gas limit, gas price, value and some others. It will override the options in pre-formed transaction.
    password
